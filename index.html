PROJECTS :

  • Time Converter Project

    • In this project, I developed a web application that converts the local time in my country to the time in more than 24 other world capitals. The application uses HTML, CSS, and JavaScript to provide a simple and functional interface. For time data management, I created a JavaScript object where each key represents a capital city, and the corresponding value indicates the time difference between my country and that city. The time zone offsets were extracted from reliable and up-to-date sources, ensuring maximum accuracy in conversions. This feature allows users to instantly check the time in multiple cities worldwide, making it easier to manage time zones when planning trips or international meetings.
